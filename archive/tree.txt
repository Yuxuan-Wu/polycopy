polycopy/
├── main.py                     # Main entry point (启动程序)
├── config.yaml                 # Configuration (配置文件 - 需要编辑监控地址)
├── requirements.txt            # Python dependencies
├── setup.sh                    # Setup script
├── verify_system.py            # System verification script
├── test_connection.py          # RPC connection tester
│
├── README.md                   # 完整文档
├── QUICKSTART.md              # 快速入门
├── PROJECT_SUMMARY.md         # 项目总结
├── .gitignore                 # Git ignore rules
│
├── src/
│   ├── __init__.py
│   ├── rpc_manager.py         # RPC节点管理（多节点冗余+自动故障切换）
│   ├── database.py            # 数据库管理（SQLite + CSV导出）
│   └── monitor.py             # 交易监控逻辑（核心监控引擎）
│
├── data/                       # 数据目录（自动创建）
│   ├── trades.db              # SQLite数据库（自动创建）
│   └── trades.csv             # CSV导出文件（自动创建）
│
└── logs/                       # 日志目录（自动创建）
    └── polycopy.log           # 应用日志（自动创建）

核心文件说明：
• main.py - 程序入口，启动监控系统
• config.yaml - 配置文件，在这里设置要监控的3个地址
• src/rpc_manager.py - 管理多个Polygon RPC节点，自动故障切换
• src/monitor.py - 监控Polymarket交易的核心逻辑
• src/database.py - 存储交易到SQLite，自动导出到CSV
• data/trades.csv - 交易数据CSV文件（可用Excel打开）
